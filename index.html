Estrutura básica do HTML
Topo

Nesta aula, aprendi a criar pastas e arquivos no VScode, utilizando a terminação ".html" e a nomenclatura convencional "index.html" para a página principal do site.

Sobre a estrutura do HTML, ele se divide em 3 partes:

  1. A tag html
  2. A tag Head
  3. A tag Body
A tag HTML compreende toda a página html
A tag Head representa configurações e efeitos carregados antes da exibição do site.
A tag Body é o que vai aparecer no corpo do site.

Falando sobre tags
Topo

As tags são comandos que indicam ao navegador a estrutura do site.

Praticamente todas precisam ser abertas ("< >") e fechadas ("< / >"), enquanto algumas não tem fechamento, como as tags input e img, que tem seu uso acompanhado com atributos, que serão melhor detalhados a seguir.

Atributos das Tags
Topo

Os atributos das tags são propriedades que uma tag pode ter. Os atributos se localizam dentro da tag, podendo ser atributos mais gerais/globais e atributos específicos.

Os atributos globais podem dar nome (name), dar um fator identificador (id) ou para dar comandos CSS(style). Esses são apenas alguns deles.

Já os atributos específicos são utilizados em conjunto com certas tags, tendo como propósito modificar o funcionamento daquela tag. Temos como exemplo o input, que necessita do "type" para determinar que tipo de campo será criado; no caso da tag img, há o atributo "src", que recebe o endereço da imagem.

Listas ordenadas e não ordenadas
Topo

Uso de listas, como listas de tópicos. Elas podem ser Ordenadas/enumeradas, como a tag ol (Ordened List) e não enumeradas/não ordenadas com a tag ul (Unordened List).

Estas tag necessitam do uso que a tag filha li (List Item) seja usada em seu interior para o desígnio dos itens da lista.

Links
Topo

Aqui começamos a "navegar" realmente. Com o uso de links vamos transitar entre páginas. Usaremos a tag a
, a de "anchor", âncora.

"É como se você ao "navegar" e aportar numa costa/ilha (que seria um site), você ancora neste lugar.
A tag a necessita de um atributo, o "href", que vai receber o endereço/url Ex: < a href="dio.me" > Texto clicável do link < / a >
Por padrão, o link vai ser aberto na mesma aba do navegador. Para abrir em outra aba, é preciso usar outro atributo, o target= "_blank"

Também é possivel usar o link para viajar nos arquivos html de uma mesma pasta. Basta colocar seu nome no href.
Também é usado para "ancoragem, que é navegar por links numa mesma página. Nesse caso é necessário que a tag destino desse link tenha um atributo id. Com isso, coloca-se no href com um "#" na frente do id.
Ex: Criei uma tag p com id="x"
< a href="#x" > Texto clicável< / a >
